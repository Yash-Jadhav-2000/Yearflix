🔥 Firebase Setup

Paste your Firebase project credentials. Saved in localStorage on this device only — never sent anywhere else.

Find credentials in Firebase Console → Project Settings → Your Apps → Web App → SDK config.

Firestore rules (Database → Rules):
allow read, write: if true;

Storage rules (Storage → Rules):
allow read, write: if true;

Firestore data structure used:
images / {year} / photos / {docId} → { url, name, uploadedAt }

📤 Upload Images

Pick a year, select your images, and hit Upload. Files go to Firebase Storage and URLs are saved in Firestore — no listAll() needed (Spark-plan safe ✅).
